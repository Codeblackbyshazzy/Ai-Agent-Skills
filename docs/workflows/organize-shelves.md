# Organize Shelves

Use `curate` when a skill is in the wrong place or the note is weak.

Examples:

```bash
npx ai-agent-skills curate frontend-design --branch Implementation
npx ai-agent-skills curate frontend-design --why "I use this when I need stronger frontend execution patterns."
npx ai-agent-skills curate frontend-design --collection build-apps
npx ai-agent-skills curate review
```

The main fields to keep sharp are:

- `workArea`
- `branch`
- `whyHere`
- `notes`
- `tags`
- `labels`
- collection membership

Keep the shelf map small and legible. If the branch feels generic, fix the branch instead of adding more vague tags.
